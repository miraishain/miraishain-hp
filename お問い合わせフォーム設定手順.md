# お問い合わせフォーム（Googleフォーム連携）設定手順

サイトのデザインはそのまま、送信内容を **Googleフォーム** に飛ばす設定です。
回答はGoogleフォーム／スプレッドシートに溜まり、**新着があると `miraishain.llc@gmail.com` 宛にメール通知**が届きます。

設定するのは `script.js` の先頭にある `GOOGLE_FORM` の部分だけです。

> ⚠️ 重要：必ず **`miraishain.llc@gmail.com` でGoogleにログインした状態**でフォームを作成してください。
> 通知メールは「フォームの所有者アカウント」に届くため、別アカウントで作ると届きません。

---

## STEP 1. Googleフォームを作る

1. https://forms.google.com を開き、空白のフォームを新規作成
2. 次の4つの質問を**この順番・形式**で作成します

| 質問タイトル | 形式 |
|---|---|
| お名前 | 記述式 |
| メールアドレス | 記述式 |
| 興味のあるサービス | ラジオボタン（選択肢を下記の通り） |
| お問い合わせ内容 | 段落 |

「興味のあるサービス」の選択肢は、サイトの選択肢と**完全に同じ文字**にしてください:

```
両方とも気になる
受注データ連携サービス
補助金検索サービス
まだ決めていない
```

> ※ 文字が1文字でも違うとGoogle側で記録されません。コピペ推奨です。

---

## STEP 2. フォームID（formId）を調べる

1. フォーム右上の「送信」→ リンク（🔗）タブを開く
2. 表示されるURLの形は次のとおり:

```
https://docs.google.com/forms/d/e/1AbCdEfG.....XyZ/viewform
                              └──────── ここが formId ────────┘
```

3. `/d/e/` と `/viewform` の間の文字列を `script.js` の `formId` に貼り付けます。

---

## STEP 3. 各項目のID（entry.数字）を調べる

1. フォーム編集画面の右上「⋮」→ **「プレビュー」**（目のアイコン）を開く
2. ブラウザでプレビュー画面を右クリック →「ページのソースを表示」
3. `Ctrl + F` で `entry.` を検索すると、各質問に対応する番号が見つかります

   例: `entry.1234567890`

   上から順に「お名前・メールアドレス・興味・お問い合わせ内容」の4つ分あります。

> もっと簡単な方法：プレビュー画面で適当に入力して送信 →
> URLに `entry.1234567890=入力値` の形で出るので、それで対応を確認できます。

---

## STEP 4. script.js に貼り付ける

`script.js` 冒頭の以下を書き換えます（`PASTE_…` を実際の値に置換）:

```js
const GOOGLE_FORM = {
  formId: '1AbCdEfG.....XyZ',          // STEP2 の値
  fields: {
    name:     'entry.1111111111',      // お名前
    email:    'entry.2222222222',      // メールアドレス
    interest: 'entry.3333333333',      // 興味のあるサービス
    message:  'entry.4444444444'       // お問い合わせ内容
  }
};
```

保存してページを再読み込みすれば、フォーム送信がGoogleフォームに記録されます。

---

## メール通知を `miraishain.llc@gmail.com` に届ける（必須）

1. `miraishain.llc@gmail.com` でGoogleにログインしていることを確認
2. Googleフォーム編集画面 →「**回答**」タブを開く
3. 右上の「**⋮**」→ **「新しい回答についてのメール通知を受け取る」** をクリックしてON

これで、フォーム送信があるたびに `miraishain.llc@gmail.com` に通知メールが届きます。

- 回答を一覧で見たい場合は、同じ「回答」タブの **スプレッドシートアイコン** で表に出力できます
- 別のアドレスにも届けたい場合は、スプレッドシート連携＋通知ルールで追加できます（ご希望あればご案内します）

---

## 動作確認

1. サイトのフォームから自分でテスト送信
2. 「送信ありがとうございます。」が表示される
3. Googleフォームの「回答」タブに反映されていればOK

> 設定前（`PASTE_…` のまま）は、送信してもお礼表示だけのデモ動作になります。
> 値を入れて初めて実際に送信されます。
